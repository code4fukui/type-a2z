# type-a2z

アルファベットのタイピング速度をテストし、向上させるためのシンプルなWebアプリケーションです。日本語のタイトル「めざせ3秒」の通り、AからZまでを3秒未満で打ち切ることを目標としています。

インターフェースには、現在の文字を大きく表示する画面と、視覚的なフィードバックを提供するレスポンシブなオンスクリーンQWERTYキーボードを備えています。

[**ライブデモ**](https://code4fukui.github.io/type-a2z/)

## 機能

- **シンプルなチャレンジ:** AからZまでのアルファベットを順番にできるだけ早く入力します。
- **視覚的なガイド:** 次に押すべきキーが仮想キーボード上で赤くハイライトされます。
- **即時のフィードバック:** 完了すると、合計時間が秒単位で表示されます（例: `3.141sec`）。
- **シンプルな操作:**
  - いつでも `Esc` キーを押すとチャレンジを最初からやり直せます。
  - 完了後に `Enter` キーを押すと再挑戦できます。
- **カウントダウン:** 各挑戦の前に3秒間のカウントダウンがあり、準備を整えることができます。

## 使い方

1. **アプリを開く:** [**ライブデモ**](https://code4fukui.github.io/type-a2z/) にアクセスします。
2. **カウントダウンを待つ:** 「3, 2, 1...」の後にチャレンジが開始されます。
3. **A-Zを入力:** ハイライトされたキーに従ってアルファベットを入力します。
4. **再挑戦:** 時間が表示された後、`Enter` キーを押して再挑戦します。

このアプリをローカルで実行するには、リポジトリをクローンし、最新のWebブラウザで `index.html` を開いてください。
