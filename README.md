# type-a2z

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A minimalist web application to test and improve your speed at typing the alphabet. The goal, inspired by its Japanese title (めざせ3秒), is to complete the A-Z sequence in under 3 seconds.

The interface features a large display for the current character and a responsive on-screen QWERTY keyboard that provides visual feedback.

[**Live Demo**](https://code4fukui.github.io/type-a2z/)

## Features

-   **Simple Challenge:** Type the letters A through Z in order as quickly as possible.
-   **Visual Guidance:** The next key you need to press is highlighted in red on the virtual keyboard.
-   **Instant Feedback:** Your total time is displayed in seconds (e.g., `3.141sec`) upon completion.
-   **Simple Controls:**
    -   Press `Esc` at any time to restart the challenge.
    -   Press `Enter` after finishing to try again.
-   **Countdown:** A 3-second countdown gets you ready before each attempt.

## How to Use

1.  **Open the app:** Visit the [**Live Demo**](https://code4fukui.github.io/type-a2z/).
2.  **Wait for the countdown:** The challenge begins after "3, 2, 1...".
3.  **Type A-Z:** Follow the highlighted keys and type the alphabet.
4.  **Retry:** After your time is displayed, press `Enter` to try again.

To run this app locally, clone the repository and open `index.html` in any modern web browser.

## Dependencies

-   [key-board](https://github.com/code4fukui/key-board/): A web component for the interactive virtual keyboard.
